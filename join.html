Requirements

What we're
looking for

We're looking for developers with at least an intermediate level in any of the following languages or tech stacks:

Python Java Kotlin C HTML / CSS / JS + Web Frameworks + Other web-related tech

You don't have to be an expert. If you can read, write, and debug code in any of these, you're good to go.

Heads up about how we work

ScriptGuild is a volunteer organization. We do not pay salaries. Participation in projects is encouraged but never forced. Think of it like open-source: you contribute when and how you want to, while building your skills and portfolio in the process.
